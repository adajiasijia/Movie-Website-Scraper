✨ 项目简介
本项目是一个轻量化的网页数据采集工具，针对电影类网站设计，可自动爬取电影名称、海报、评分、导演、主演、简介等核心信息，并将数据导出为 CSV 格式，方便后续数据分析与可视化。
🎯 核心功能
定向数据爬取：精准解析电影网站页面，提取影视核心信息字段
数据清洗与格式化：统一处理时长、日期等格式，输出规范数据
批量采集：支持多页面遍历，高效获取榜单类数据
持久化存储：自动导出为 UTF-8 编码的 CSV 文件，便于二次处理
容错机制：内置请求超时与空值处理，提升程序稳定性

🛠️ 技术栈
Python	核心开发语言
requests	发送 HTTP 请求，获取网页源码
lxml	解析 HTML/XML 文档，提取目标数据
csv	生成并保存结构化 CSV 文件
re	正则表达式，用于文本格式清洗

🚀 快速开始
安装依赖

pip install requests lxml
python tmdb_top_movies_crawler.py
